# Digital Privacy Score Checker

## Description
This project evaluates a user's digital privacy based on their online habits and provides a privacy score along with suggestions for improvement.

## Features
- Privacy score calculation
- Risk level classification
- Personalized recommendations

## How to Run

1. Clone the repository:
   git clone <your-repo-link>

2. Navigate to the project folder:
   cd privacy-score-checker

3. Run the program:
   python main.py

## Example Inputs
- Password strength: strong
- 2FA: yes
- Public WiFi: no
- Permissions: medium

## Output
- Privacy Score
- Risk Level
- Suggestions
