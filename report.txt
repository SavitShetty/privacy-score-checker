🔹 Introduction

In today’s digital age, users frequently interact with online platforms without fully understanding the risks to their personal data. Lack of awareness regarding security practices such as strong passwords, two-factor authentication, and safe internet usage increases vulnerability to cyber threats. This project aims to address this issue by providing a simple tool to evaluate and improve digital privacy.

🔹 Problem Identification

Many users unknowingly compromise their privacy due to poor digital habits like weak passwords, excessive app permissions, and frequent use of public WiFi. Existing tools are often complex and not beginner-friendly, creating a gap between user behavior and privacy awareness.

🔹 Proposed Solution

The project introduces a Digital Privacy Score Checker that analyzes user inputs related to their online habits and generates a privacy score between 0 and 100. It also categorizes users into risk levels and provides personalized suggestions to improve their privacy practices.

🔹 Methodology

The system collects user inputs such as password strength, 2FA status, public WiFi usage, and app permissions. A scoring algorithm assigns weights to each factor and calculates a final score. Based on this score, users are classified into High Risk, Moderate Risk, or Safe categories. Suggestions are generated accordingly to guide users toward better security practices.

🔹 Implementation

The project is implemented in Python using a modular structure. Separate files handle input processing, score calculation, and suggestion generation. The application runs through a command-line interface and is managed using version control on GitHub.

🔹 Challenges Faced

Key challenges included designing a fair scoring system, handling user inputs effectively, maintaining modular code structure, and managing GitHub integration. Debugging and testing also required careful attention to ensure accurate results.

🔹 Learning Outcomes

The project enhanced understanding of real-world problem solving, Python programming, modular design, and version control using Git. It also increased awareness of cybersecurity practices and improved debugging skills.

🔹 Conclusion

The Digital Privacy Score Checker successfully provides a simple and effective way to evaluate and improve digital privacy. It demonstrates the application of programming concepts to solve real-world problems and lays a foundation for future enhancements in cybersecurity tools.
